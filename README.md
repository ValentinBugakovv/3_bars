# Ближайшие бары

Программа расчитывает самый большой, маленький, близкий бар в г.Москва. Список московских баров в формате JSON можно взять [здесь](https://devman.org/media/filer_public/95/74/957441dc-78df-4c99-83b2-e93dfd13c2fa/bars.json). или с сайта data.mos.ru. 

# Как запустить

Скрипт требует для своей работы установленного интерпретатора Python версии 3.5

Запуск на Linux:

```bash

$ python bars.py # possibly requires call of python3 executive instead of just python
```

Запуск на Windows происходит аналогично.
